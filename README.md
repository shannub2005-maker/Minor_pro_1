# 📊 WhatsApp Group Chat Analyzer

A comprehensive **WhatsApp Chat Analysis** project built using **Python** and **Google Colab** to extract meaningful insights from exported WhatsApp group chats. This project performs data cleaning, preprocessing, exploratory data analysis (EDA), and visualization to understand user activity, messaging behavior, and group engagement.

# 🚀 Features

* 📂 Upload and analyze exported WhatsApp chat files (`.txt`)
* 🧹 Clean and preprocess raw chat data
* 👤 User-wise message statistics
* 💬 Total messages, words, media, and links shared
* 📅 Daily, weekly, and monthly activity analysis
* 📈 Timeline visualizations of chat activity
* ⏰ Heatmaps showing the most active hours and days
* 😊 Emoji usage analysis
* ☁️ Word Cloud generation for frequently used words
* 🔥 Most active participants and contribution analysis
* 📊 Interactive charts and graphs using Matplotlib and Seaborn

# 🛠️ Technologies Used

# 📁 Dataset

The project uses an exported **WhatsApp group chat (.txt)** file. Personal information can be anonymized before analysis to maintain privacy.

# 📊 Project Workflow

1. Import WhatsApp chat data
2. Data cleaning and preprocessing
3. Feature engineering
4. Exploratory Data Analysis (EDA)
5. Statistical analysis
6. Data visualization
7. Insights generation

# 📌 Key Insights

* Identify the most active members in the group.
* Discover peak chatting hours and busiest days.
* Analyze emoji usage patterns.
* Generate word clouds of frequently used words.
* Track chat activity over time.
* Compare user participation and engagement.
